Movie API

The following API contains a database of movies allowing users to access movie insights such as the Name, Genre, Directors and Cast.

Description endpoints:

> > > > > > > > > > > > > > > > > > >
Request URL Method Body data Response
Welcome to my FILM list '/' GET Display Welcome Page to the user.
List Movies/movies GET Return a list of ALL movies to the user
Movie Title/movies/title GET Return data about a single movie by title to the user
Movie Genre/movies/genre GET Return data about a genre
Directors list/directors/directors name GET Return data about a director by name
Register user/users POST Allow new users to register
Update info/users/username PUT Allow users to update their user info
Favourite movie/users/username/favorites/movie POST Allow users to add a movie to their list of favorites
Remove Movie from Favourites/users/username/favorites/movie DELETE Allow users to remove a movie from their list of favorites
Deregister user/users/username DELETE Allow existing users to deregister
